Nome: Mauricio Lindemann

Problema: Algoritmo de Peterson

Descrição: É um algoritmo de programação concorrente para exclusão mútua, que permite a dois ou mais processos ou subprocessos compartilharem um recurso sem conflitos, utilizando apenas memória compartilhada para a comunicação.A solução original de peterson foi formulada para trabalhar originalmente com apenas 2 processos, ao longo dos anos foi aperfeiçoada, é uma solução a  nível de software simples e eficaz mas que não é vastamente utilizada devido a suas limitações para trabalhar com um alto número de processos.
